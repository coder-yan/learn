# 常用快捷键
```code
临时将pannel全屏（重复一次恢复）
ctr-b z
垂直分割pannel
ctr-b %
水平分割pannel
ctr-b "
列出window窗口
ctr-b w
列出pannel窗口的ID，按id可以切换pannel
ctr-b q
重命名窗口
ctr-b ,
切换pannel
ctr-b h/j/k/l
切换window窗口
ctr-b shift-h/l
退出tmux模式（tmux没有退出）
ctr-b d
```
